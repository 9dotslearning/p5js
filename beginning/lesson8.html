Spirals

In this lesson, you'll learn how to make beautiful spirals by changing the rotation, position, and size of different shapes.

1. Change Position

Try these challenges: Change exactly one number in the script to make the spiral longer. Notice that in the script, the ellipse's x- coordinate (sideways position) increases by 1 over and over. On screen, that means the ellipse moves further and further from the center over time. Change the script so x increases by different amounts, clicking ▶Play after each change to see the effect. Which values added to x make spirals that you like? The variable r increases by 0.1 over and over. Change the script so that r increases by other numbers. Which values added to r make spirals that you like? Just for fun, change the height and width in the ellipse command to make long, flat ellipses or tall, skinny ones instead of circles. Just for fun, fill the ellipses with your favorite color.

Here's a slow-motion image of an ellipse whose x-coordinate (sideways position) gets larger and larger as the canvas rotates. The effect is to create a spiral, the same shape as star galaxies in space.

To make the spiral longer, change line 13 to a number bigger than 10, like this:
if (r > 20) noLoop();
This change allows the canvas to make more rotations before it stops, so the spiral has a chance to grow longer.

To increase x by different amounts, include different numbers on line 16 — for example, change 0.5 to 1.2:
x = x + 1.2;
When you click ▶Play, you'll see a wider spiral because now the ellipse travels 1.2 pixels (instead of 0.5 pixels) on each rotation.

To increase r by different amounts, include different numbers on line 12 — for example, change 0.1 to 0.01:
r = r + 0.01;
When you click ▶Play, you'll see a flatter spiral. The variable r changes how much (or little) the canvas rotates each time that function draw() runs, with a wide variety of different effects. In this case, the rotation is 0.01 (instead of 0.1), meaning that the canvas rotates less each time. As the ellipse moves sideways, it rotates down less than it used to, making the spiral curve less. Tip! While you're changing r, you may also wish to change line 13 to allow more (or fewer) rotations before the animation stops.

2. Change Size

Tinker

Click ▶Play. The animation is created by a single ellipse. Let's investigate how it's done.
Try these challenges: As the canvas spins, each new ellipse becomes a little wider. What part of the code changes the width of the ellipse? Can you change the code so that the ellipses get wider a little more slowly? a little more quickly? Which animations do you like? Can you change the code so that the rotation changes a little more slowly? a little more quickly? Which animations do you like?

On line 1, we create a variable named sz that gets 0. On line 12 we use that variable to define the width of the ellipse:
ellipse(0, 0, sz, 10);
On line 14, we change sz by 1 each time function draw() runs:
sz = sz + 1;
so the ellipse gets wider over time.

To make the ellipses get wider more slowly, include smaller numbers on line 14 — for example, change 1 to 0.5:
sz = sz + 0.5;
To make the ellipses get wider more quickly, include numbers larger than 1.

To make the canvas rotate more slowly, include smaller numbers on line 13 — for example, change 1 to 0.25:
r = r + 0.25;
The variable r determines how far the canvas rotates on line 11:
rotate(r);
So smaller numbers make less rotation.

Example #1 — Spinning Circles

For this project, see the first GIF image at the beginning of this lesson. The starter code below shows a single ellipse changing colors randomly. You must make it move further and further from the center of the canvas over time, and you must also make the canvas rotate. If you wish, you can make the animation stop after awhile. The project includes all the variables you'll need. Use the variable r to change the rotation and the variable x to change the ellipse's x-coordinate, or sideways position. 

let r = 0;
let x = 0;

function setup() {
  createCanvas(400, 400);
  colorMode(HSB);
  background(0, 100, 0);
}

function draw() {

  translate(width/2, height/2);
  rotate(r);

  fill(random(360), 100, 100);
  ellipse(x, 0, 8, 8);

}

Example #4 — Lines

For this project, see the fourth GIF image at the beginning of this lesson. The starter code below shows an ellipse at one end of a short line. You must make the ellipse and the line change their positions together, and you must make the canvas rotate. Feel free to change the colors and to stop the animation. One end of the line moves straight downwards over time, and the other end moves sideways by the same amount (making the line longer). To say this another way, one end of the line changes its y-coordinate and the other end changes its x-coordinate by the same amount. The ellipse changes its y-coordinate so that it moves with one end of the line. You can see these movements in the GIF image below.
 The canvas rotates while the line and ellipse are moving, as in the GIF image below.
 The project includes all the variables you'll need. Use the variable r to change the rotation and the variable z to change the ellipse's y-coordinate and the line's x- and y-coordinates.

let z = 0;
