Computer Programming

Computer programming means telling a computer what you want it to do in a language it understands.

Computers understand hundreds of languages — JavaScript, Python, Processing, Swift, C#, C++, and many more. Each language has unique strengths. The language you'll learn in these lessons, p5.js, is especially good at making web-based creative projects that involve drawing and sounds, but it has many other capabilities as well.

p5.js is closely related to several other languages. It's a specialized version (called a library) of JavaScript, which makes web pages that respond to clicks, mouse movements, time, date, location, and many other events. In addition, p5.js is related to Processing which, like p5.js, makes creative projects. But Processing projects are more difficult to create and change on the web, so we'll focus on p5.js instead.

Code

In these lessons, the words code, program, script, sketch, and commands all mean the same thing — a series of instructions that tell a computer what to do. Here's a typical p5.js sketch:

createCanvas(250, 200);
background(0);

createCanvas(250, 200) creates a window where you see drawings, called the canvas. background(0) paints the window black. You'll learn more about these commands and many others in the lessons that follow, a little at a time.

When the computer actually performs these commands, we say that it runs, executes, or plays them. Computers run one command at a time, in order. If the computer doesn't understand a command, it usually stops and shows an error message; the commands after the error often don't run at all until you fix things.
