Lesson 2

Create an Account

To make new projects and save them, you can create an account at the p5.js Web Editor. Follow the steps below.

← Back to Home

  • Open the the p5.js Web Editor in a new tab.
  • Click Sign up at the top right.
  • Complete the sign up form.
  • To verify your email address, sign into the email you used when you filled out the form, open the message that looks like the image below, and click the Verify Email button inside.

      Troubleshooting Tips

    • If you can't find the verification email, check your Spam folder.
    • If there's still no email, go back to the p5.js Web Editor. Click your name at the top right, and then click Settings. In the next window, click Resend confirmation email and try again.
    • If you got the email and opened it but don't see the Verify Email button, find the link provided inside the email, and then copy and paste it into your web browser instead.
  • Go to the p5.js Web Editor and sign in.

  • After signing in, you'll see a screen like the one below.
  • Name your project by clicking the pencil icon. Type a sensible name like practice and then press the enter or return key.
  • Immediately, save the project! Click the File menu and then click Save in the drop-down menu. After you save the first time, the p5.js editor will automatically save any additional work on this project about every 30 seconds.
  • Click the Play button to run the code. In this case, you'll see a gray background on the right side of the screen, which was created by the commands you see at the left.
  • We'll write more code in a moment.
