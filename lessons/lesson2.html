createCanvas defines the width and height of the canvas, the drawing area that you see after you click ▶Play. Its size is measured in pixels. A pixel is the smallest dot that the screen can show; it's about 1/100 inch wide and tall. This canvas is 250 pixels wide and 200 pixels tall.

background changes the color of the canvas.

The numbers in parentheses are called arguments, and they change how commands work. For example, background(0) creates black and background(255) creates white. Arguments between 0 and 255 produce gray, with lighter grays as arguments get bigger.

When you click Click ▶Play, we say that you run the script.

If things get messed up after you make changes, you can restore the starting code by clicking Revert.

3. Color

Let's add color to our drawing!

Tinker

Click ▶Play. You'll see a yellow background. It's created by the commands colorMode(HSB) and background(51, 100, 100). You can learn more about these commands by clicking Color Help below. To add your own colors, go to HSB Color Picker, adjust the sliders, and then copy the 3 numbers for your color into background below. Be sure to copy them in order! And remember to click ▶Play. ðŸ™‚

At HSB Color Picker, notice the 3 numbers labeled Hue, Saturation, and Brightness. These 3 qualities, taken together, create a color.

Hue is the basic color — red, gold, turquoise, and so on. It is represented as degrees around a color circle. For example, our yellow is at 51°.

Saturation is the intensity of the color. It is represented as a percentage, with 100% meaning full color and 0% meaning no color (white).

Brightness is the shade or amount of black mixed into color. It is represented as a percentage, with 100% very bright (no black) and 0% meaning the color has become completely black.

Location on Screen

In ellipse(150, 100, 50, 35), the first number, 150, is the ellipse's distance from the left edge of the screen. The number 100 is its distance from the top.

Here's a picture of the ellipse and distances.
These distances are so helpful for placing images on screen that we give them a special name: coordinates. A shape's distance from the left edge is called its x-coordinate. Its distance from the top is called its y-coordinate.
We write the coordinates like this:
150, 100
The x-coordinate is always first, followed by the y-coordinate. The two coordinates are separated by a comma.

8. setup() and draw()

Tinker

Click ▶Play. You'll see dozens of ellipses! Where did they come from?
Notice that the script has been divided into two sections called function setup() and function draw(). The commands in function setup() happen once, as soon as you click ▶Play. But the command in function draw() happens over and over, endlessly, creating infinite ellipses at random positions.
Try these challenges: Change the ellipse command to draw ellipses with random sizes. Draw randomly-colored ellipses by adding commands in function draw(). Be sure to add them above the ellipse command.

Horizontal whitespace before or after commands doesn't matter. Vertical whitespace doesn't matter. The order does matter. function setup() { } does matter. Commands go inside { }, like in a little suitcase. setup() must be first, and the commands you learned so far must be inside one of these functions you can have many commands in draw(), and they all happen endlessly
