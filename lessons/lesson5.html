In this lesson, you'll learn how to create variables and use them to move shapes across the screen.


1. Many Rectangles Falling

Tinker

Click ▶Play. You'll see many rectangles that appear one after the other, from top to bottom.
Try these challenges: In the first line of code
p = 0;
change 0 to a bigger number between 0 and 250. What happens? Click Revert to restore the previous script. Then change the script to make the rectangles appear further apart. Change the script to make the rectangles appear in the opposite direction, from bottom to top. Change the label p to another letter. Be sure to change p to the same, new letter in all 3 places it appears in the script. What happens?

6. Changing Colors

Tinker

Click ▶Play. There are two yellow rectangles that both turn pink. You might also notice the new code on line 17: if (y > 200) c = 325;. Let's investigate.
Try these challenges: On line 17, change 325 to another number between 0-360. Then click ▶Play. What happens? Switch the order of the two lines of code at the end of the script so they look like this:
if (y > 200) y = 0;
if (y > 200) c = 325;
Click ▶Play. The rectangles don't change color. Why? Click Revert to restore the previous script. Then change line 17 again so that c gets a random number between 0-360. Then click ▶Play. What happens? Why?
