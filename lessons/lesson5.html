Rotates about the center counter-clockwise. The number 6 is a little more than a full turn. p5.js is using a unit of turning called a radian, which is about 57 degrees or, more precisely, 180/PI degrees.

8. Transparency

Tinker

Click ▶Play. Each ellipse gets bigger and moves away from the center. Also, the drawing appears gray at the edges and darker at the center.
There are 2 new commands:
◆ noFill() makes the inside of ellipses transparent. ◆ stroke(0, 100, 0, 0.2) now has a fourth argument. This argument makes the stroke (borders) partially transparent, so they appear faded.

Try these challenges: In stroke, change the decimal 0.2 to other decimals between 0 and 1. Find a decimal you like. Make the stroke a color instead of black and gray. Replace the stroke and fill commands with the two new commands below. What happens?
fill(330, 100, 100, 0.1);
noStroke();

TBD
The command noFill(); makes fill completely transparent, so you can see ellipses behind other ellipses. In previous drawings, each new ellipse obscured previous ellipses.
There's also a way to change transparency a little at a time (instead of turning it on or off completely). Notice that the stroke command includes fourth argument: stroke(0, 0, 100, 0.2)
This argument is called alpha. Alpha controls the amount of transparency a little at a time. An alpha of 0 is completely transparent, and an alpha of 1 is not at all transparent. You can set alpha to any number between 0 and 1.

9. Examples

Copy each example below into the p5.js Web Editor, completely replacing any code that's already there. Play with the code to find effects that you like!

Example #1 

let r = 0;

function setup() {
  createCanvas(400, 400);
  colorMode(HSB);
  background(0, 100, 0);
}

function draw() {
  translate(200, 200);
  rotate(r);
  stroke(r, 0, 100, 0.2);
  strokeWeight(2);
  line(-r, 0, r, 0);
  r = r + 0.5;

  if (r > 200) noLoop();
}

  • line(-r, 0, r, 0);
    Creates a line centered in the middle of the screen. The first argument starts the line r pixels to the left of center. The third argument ends the line r pixels to the right of center.
  • if (r > 200) noLoop();
    stops the animation when r is greater than 200.

Example #2 

let r = 360;

function setup() {
  createCanvas(400, 400);
  colorMode(HSB);
  background(0, 100, 0);
}

function draw() {
  translate(200, 200);
  rotate(r);
  fill(r, 100, 100);
  rect(0, 0, r/2, r/2);
  r = r - 1;

  if (r < 0) noLoop();
}

  • rect(0, 0, r/2, r/2);
    r/2 means "r divided by 2"
  • r = r - 1;
    subtracts 1 from r each time function draw() repeats, making r smaller and smaller over time.
  • if (r < 0) noLoop();
    stops the animation when r is less than 0.
