In this lesson, you'll learn how to make shapes do something and then undo it. For example, you'll make shapes change direction when they touch the edge of the canvas. You'll learn how the background command and variables are essential for making these back-and-forth animatioins.

2. One Ellipse

Tinker

Click ▶Play. You'll see one ellipse traveling across the screen from left to right. The code almost the same as in the previous step, but there's one difference. Can you find it?
Try these challenges: Delete the background command. What happens? Why? Then click Undo to restore the background command. Then click ▶Play. What happens? Why?

Explanation

Here's why the position of the background command matters. When the background command is in function draw(), the background color is painted on the screen over and over, once each time that function draw() runs. That means the background is painted about 60 times every second! Each time the background is painted, it covers up anything that was previously drawn. So you see just one ellipse at a time, moved a little to the right from where it was previously. On the other hand, when the background command is in function setup(), the background is painted just once when the project begins. The ellipses painted by function draw() — which runs over and over — all stay on screen.

Example #1 — Bouncing Circles

For this project, see the first GIF image at the beginning of this lesson. The yellow circle moves straight across the screen and back again. The other two circles move diagonally. The two diagonally-moving circles reflect each other's movement, like the shapes in the Reflections lesson. Feel free to make the same animation or something very different! The example includes all the variables you'll need, although you may wish to add or change them; many different strategies work successfully! The code contains just one ellipse, so you may want to add more. The ellipse has no fill and a big stroke weight that makes a thick outline. This outline is partially transparent because of the fourth argument in the stroke command, 0.2. 

/* *** BOUNCING CIRCLES *** */

let x = 0;
let xstep = 1;
let y = 0;
let ystep = 2;

function setup() {
  createCanvas(400, 400);
  colorMode(HSB);
}

function draw() {
  background(0, 0, 100);

  noFill();
  strokeWeight(50);

  stroke(105, 100, 100, 0.8);
  ellipse(x, y, 200, 200);


}

Example #2 — Colored Squares

For this project, see the second GIF image at the beginning of this lesson. A large square fades out and fades back in, over and over. A small square inside the large one does the opposite — it fades in and fades back out, over and over. Feel free to make the same animation or something very different! The example includes all the variables you'll need, although you may wish to add others; many different strategies work successfully! The example code also includes only the bigger square, so you'll need to add the smaller one. The stroke command for the bigger square includes transparency which is changed over time by the variable t. This variable should change from 1 (the square is completely visible) to 0 (the square is completely transparent) and back again endlessly. The transparency for the smaller square should also be changed by a variable. Can you figure out how this variable can make the smaller square to begin invisibly and become visible? 

/* *** COLORED SQUARES *** */

let t = 1;
let tstep = -0.01;

function setup() {
  createCanvas(400, 400);
  colorMode(HSB);
}

function draw() {
  background(0, 100, 0);

  noFill();
  stroke(344, 100, 100, t);
  strokeWeight(100);
  rect(0, 0, 400, 400);

  t = t + tstep;


}

Example #3 — Grow and Shrink

For this project, see the third GIF image at the beginning of this lesson. A large ellipse at the top shows its lower half on the canvas (the other half is there but can't be seen above the canvas). There's a similar large ellipse at the bottom, and they grow and shrink together. Three smaller ellipses at the center of the canvas grow and shrink more quickly. Feel free to make the same animation or something very different! The example includes only some of the variables you'll need, so you'll need to add more. The example also includes only the two large ellipses, so you'll need to add the smaller ones. 

/* *** GROW AND SHRINK *** */

let sz = 120;
let szstep = 1;

let running = false;

function setup() {
  createCanvas(400, 400);
  colorMode(HSB);
}

function draw() {
  background(30, 10, 30);
  strokeWeight(24);

  noFill();
  stroke(105, 100, 100);
  ellipse(200, 0, sz, sz);
  ellipse(200, 400, sz, sz);


}
