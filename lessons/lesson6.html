5. Color Waves

Tinker

Click ▶Play. All the shapes are red.
Try these challenges: In the first line of code, we set variable c to 0. Then we used c in the fill command on line 11 to choose the hue. When the hue is 0, all shapes are red. Change c to a different number in the first line of code. Click ▶Play. Find a number for the hue that you like. Type a new line of code that makes c change endlessly. Then click ▶Play. You should see waves of color, but you'll notice that they stop after a short time. Why? The biggest hue is 360. Type a new line of code so that when c is greater than 360, it goes back to 0 and starts over. You can make some lovely drawings with these commands. Try making the color waves very slow, and add new shapes. Make each shape a different color. Have fun!
